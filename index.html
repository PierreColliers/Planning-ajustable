Publiez ce fichier HTML gratuitement en 3 étapes via GitHub Pages — aucune compétence technique requise.

1
Téléchargez le fichier HTML en cliquant sur « Exporter SVG » puis utilisez ce fichier. Ou téléchargez directement cette page avec Ctrl+S / Cmd+S.
planning-immobilier.html
2
Créez un compte gratuit sur github.com, puis créez un nouveau dépôt public. Déposez-y votre fichier HTML en le renommant index.html.
3
Activez GitHub Pages : dans les Settings du dépôt → Pages → Source : main / root. Votre lien sera : https://votre-nom.github.io/planning-immobilier/
✓
Astuce : utilisez le bouton « Sauvegarder le lien » pour encoder votre planning dans l'URL — partagez ce lien et vos collègues verront exactement votre version.
